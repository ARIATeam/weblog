<a href="http://habtm.com/articles/2005/09/16/weed">Weed</a> is a new stats package in leu with Mint and others build using Ruby on Rails. It's open source and rapidly improving. Do check it out.

<a href="http://mir.aculo.us/articles/2005/09/18/pure-ruby-sparklines">Part of Weed</a> is a <a href="http://collaboa.weed.rbse.com/repository/file/branches/pgsql/lib/spark_pr.rb">pure-Ruby Sparklines implementation</a> by Thomas Fuchs that's meant to be API-compatible with Geoffrey's Sparklines gem (that requires RMagick).
